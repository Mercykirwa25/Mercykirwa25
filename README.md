## 📂 Portfolio Projects

### 🏆 Customer Churn Prediction — Telecom
> **Best model: Tuned XGBoost · 96% Accuracy · 82% Recall · 90% ROC-AUC**

Predicted customer churn for ABC Telecommunication Company (26% churn rate, ~$2.4M revenue loss) using the SyriaTel dataset. Built a full ML pipeline including EDA, feature engineering, SMOTE for class imbalance, and hyperparameter tuning across Logistic Regression, Decision Tree, Random Forest, and XGBoost models.

**Key findings:** International plans, high-day usage, and frequent service calls are the strongest churn predictors.

`Python` `XGBoost` `SMOTE` `Scikit-learn` `Pandas` `Matplotlib` `Seaborn`

### 🤖 Apple & Google Tweet Sentiment Analyzer (NLP / Deep Learning)
> **DistilBERT · 90% Recall on Negative Sentiment · Streamlit Deployment**

Built a real-time sentiment monitoring system for tech distributors using transfer learning with DistilBERT. Implemented binary classification (alerting) and multiclass classification (trend analysis). Used Focal Loss and data augmentation (synonym replacement) to handle severe class imbalance — outperforming classical ML models (XGBoost: 57% recall vs DistilBERT: 90% recall).

**Key features:** Real-time negative sentiment alerts, Streamlit dashboard, Weights & Biases experiment tracking.

`Python` `DistilBERT` `PyTorch` `HuggingFace` `Streamlit` `Weights & Biases` `TF-IDF` `GloVe`

🔗 [View Project](https://github.com/Mercykirwa25)
